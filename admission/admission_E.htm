Introduction to Current Projects

Project 1: Machine/deep learning and cybersecurity

Machine learning is a paradigm shifting technology that is widely used to solve many problems in science and engineering. On the other hand, cybersecurity is important in protecting the confidentiality, integrity, and authenticity of data. In this project, the candidate is required to use machine learning/deep learning to solve some existing problems in real world. He/she can also propose cybersecurity techniques to secure the existing machine learning/deep learning applications. Good programming skill (e.g. python) and interest in machine learning is required to undertake this project.


Project 2: Implementation of post-quantum cryptography schemes in GPU, microcontroller or FPGA

The candidate will be trained on the mathematical foundation of post-quantum cryptography (PQC) schemes, especially lattice, code and multivariate based schemes. He/she is expected to implement the selected PQC schemes in hardware platform like GPU, microcontroller or FPGA. Good programming skill (Verilog, C/C++ or assembly language) is required to undertake this project. However, knowledge to program GPU (e.g. CUDA or OpenCL) is optional.


Project 3: Efficient and secure computation on encrypted data

Fully homomorphic encryption (FHE) is a new technique that allows the computation to be carried out in encrypted domain. This is especially useful in protecting the user privacy in several areas like medical and financial information system. In this project, the candidate will be trained on the mathematical foundation of FHE schemes, which is mainly based on lattice hard problems. He/she is expected to implement the selected FHE schemes in CPU or GPU to perform a designated task (e.g. machine learning). Good programming skill (C/C++ or python) is required to undertake this project. However, knowledge to program GPU (e.g. CUDA or OpenCL) is optional.
